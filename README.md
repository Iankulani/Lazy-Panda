# Lazy-Panda

<img width="636" height="500" alt="lazy" src="https://github.com/user-attachments/assets/f0df71e2-900f-4127-bc0e-b552880259a3" />


Lazy-Panda is an advanced cyber security automation tool designed to simplify and accelerate network reconnaissance and vulnerability analysis. 
With a single command, Lazy-Panda performs multiple operations including IP analysis, traceroute, ping scanning, and Nmap-based port and service detection to provide a complete overview of network health. The tool intelligently aggregates results into clear, actionable reports, helping users quickly identify weaknesses and potential attack surfaces. Lazy-Panda is ideal for students, network engineers, and cyber security professionals who need fast, reliable insights without complex setup. Its lightweight design, accuracy, and automation make Lazy-Panda a powerful companion for ethical hacking, penetration testing, and defensive security operations.

## How to clonw the repo
```bash
git clone https://github.com/Iankulani/Lazy-Panda.py
cd Lazy-Panda
```

## How to run
```bash
python Lazy-Panda.py
```
